# Unsupervised Semantic Frame Induction via Contextual Embeddings and Graph Partitioning

## Abstract
Semantic frame induction is the task of automatically discovering event schemas and their participant roles from raw text without hand-labeled data. In this work, we propose a comprehensive pipeline that evaluates multiple approaches to unsupervised frame induction. We compare context-aware representations from pre-trained language models (BERT, MiniLM) against traditional methods (TF-IDF, SpaCy vectors) and evaluate graph-based community detection algorithms against traditional clustering methods like K-Means and Spectral Clustering. We use the FrameNet v1.7 corpus as our ground truth for evaluation.

## Introduction
A semantic frame represents an event schema, consisting of a target predicate and its semantic roles. For example, the sentence "John bought a car from Mary" evokes the `Commerce_buy` frame, with roles such as `Buyer`, `Goods`, and `Seller`. While supervised models for Semantic Role Labeling (SRL) perform well, they are strictly limited to the domains and languages of their training data. Unsupervised frame induction offers a path to discover latent semantic structures in new domains. 

This project explores:
1. Identifying predicates and their argument structures using dependency parsing.
2. Generating expressive contextual embeddings for the predicates.
3. Discovering frame labels via clustering and graph community detection.

## Related Work
Early work by Modi et al. (2012) utilized probabilistic models to induce frames. More recently, Ustalov et al. (2018) employed graph-based clustering (e.g., Chinese Whispers) on synonymy graphs, while modern approaches leverage BERT representations to cluster context-aware sense embeddings (Aharoni & Goldberg, 2020). We bridge these methodologies by directly comparing graph-based induction on predicate-argument structures with state-of-the-art dense clustering on LLM embeddings.

## Methodology

### Data Extraction
We utilize the NLTK FrameNet v1.7 API, extracting a subset consisting of the most frequent verbs and maximum 10,000 sentences to ensure computational tractability.

### Linguistic Analysis & Feature Extraction
We employ `spaCy` to parse the dependency structure of each sentence. We locate the target predicate and extract its syntactic arguments (e.g., `nsubj`, `dobj`, prepositional objects) to construct a Predicate-Argument graph in NetworkX.

### Representations
We compare four distinct vectorization strategies:
1. **TF-IDF**: Sparse baseline.
2. **SpaCy Vectors**: Static 300D Word2Vec-style embeddings representing the average sentence meaning.
3. **MiniLM**: `sentence-transformers/all-MiniLM-L6-v2`, highly optimized for semantic similarity.
4. **BERT**: `bert-base-uncased` via mean-pooling over the sequence token representations.

### Clustering Methods
For continuous representation embeddings, we evaluate:
- K-Means
- Agglomerative Clustering
- Spectral Clustering

For graph representations, we use **Louvain Community Detection** directly over the edges formed by shared predicate-argument co-occurrences.

## Experiments
We conduct formal evaluation comparing the induced clusters to the gold-standard FrameNet labels. The primary evaluation metrics are:
- **Adjusted Rand Index (ARI)**
- **Normalized Mutual Information (NMI)**
- **Cluster Purity**
- **V-Measure**

*The numerical comparison table is auto-generated by our experimentation pipeline `main.py` in the `results/comparison_results.csv` artifact.*

## Discussion & Error Analysis
Qualitative error analysis of the generated clusters (visible in the saved t-SNE files) reveals a few common failure patterns:
1. **Polysemy vs Frame Differences:** Highly polysemous words (e.g., "run") often cluster topologically close despite triggering different frames (e.g., `Self_motion` vs `Operating_a_system`). LLM embeddings help disambiguate context, but may still be dominated by syntagamatic lexical associations.
2. **Data Sparsity:** Low-frequency frames are often absorbed by larger parent clusters in K-Means.

## Conclusion and Future Work
We implemented a robust pipeline demonstrating unsupervised semantic frame induction. Modern transformer models like MiniLM significantly outperform TF-IDF and static embeddings. Future work includes utilizing contrastive learning to fine-tune the embeddings specifically on predicate similarities and replacing heuristics with full semantic dependency parsing graphs.
